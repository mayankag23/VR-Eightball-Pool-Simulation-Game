# VR-Eightball-Pool-Simulation-Game

This is a Unity-based pool game that can be played in both PC (mouse and keyboard) mode and a VR-simulated mode. The project focuses on providing a realistic pool playing experience while still allowing the game to run without a VR headset.

## Features

* Basic pool game setup with cue stick and ball interaction
* Playable in PC mode using mouse and keyboard
* VR mode simulation using keyboard controls
* Camera reset and overhead view option
* On-screen guidance for controls

## How to Run

### Using Unity

1. Open the project in Unity
2. Open the main scene
3. Press the Play button
4. The game will start in PC mode

### Using Build File

1. Open the Build folder
2. Run the `.exe` file
3. The game will launch in PC mode

## Controls

### PC Mode

* Mouse movement: Aim the cue stick
* Left mouse drag: Adjust shot power
* Space bar: Reset camera or reposition cue
* UI button (top-left): Toggle overhead view

### VR Mode (Keyboard Simulation)

Press `V` to switch to VR mode

* Arrow keys: Rotate cue stick
* W / A / S / D: Move stick
* Q / E: Move stick up and down
* G: Hit the ball
* Space bar: Reset camera

Additional:

* Y: Show control instructions
* X: Show hand/controller mode
* [ and ]: Switch interaction mode

## Notes

The project was intended to run on a VR headset, but due to compatibility and size limitations, it may not run directly on the device. All VR interactions have been mapped to keyboard controls so the game can still be demonstrated on a PC.
